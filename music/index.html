森田崇允（モリタタカマサ）

Singer-Songwriter / Guitarist 優しい声だが、つぶやくような寂しげな歌。そこに鳴るクラシック・ギター。しかしながらワールド・ミュージックやブギウギなども吸収した曲やスタイルは個性的。パーカッションやアコーディオンとのアンサンブルも温度感があって引き込まれる。(CDジャーナル データベースより) 2016年 1st Solo Album『想いひらく世界』WaikikiRecordよりリリース

公式サイト

MAHOROBA

2013年の結成から幾度かのメンバーチェンジを経て、それぞれに異なるセンスと確かなスキルを持った 七人が夜毎に酒を酌み交わし、奇跡的にまとまったBlack music集団。 「まほろば」の言葉の意味するとおり、家のような、近所の飲み屋のような、そんな居心地の良い空間を音楽で生み出す事をモットーに、二日酔いを乗り越えて活動中。

公式サイト

山口ゆきのり/ Yukinori Yamaguchi

ハモンドオルガン、キーボード奏者。1976年3月17日神奈川県生まれ。 1999年ハモンドオルガンのインストバンド、FULL SWINGのリーダーとして活動を開始。並行して、曽我部恵一PINK BANDへの参加や、サニーデイサービスのレコーディング、真心ブラザーズ、THEイナズマ戦隊のライブ参加などサポート活動、CMレコーディングなどに参加。

公式サイト

ARA

singer-songwriter ( Vocal & Keyboard ） インディーズレゲエバンドのキーボーディストとして、全国タワーレコードにてCDを発売し国内でライブ活動を行う。 Jazz、Funk、R&B、Reggae、Hip Hop、Soul などのBlackMusicを好み、 2017年にリリースした作品「NEXT」は、作詞 / 作曲 / アレンジ / Programmingまで初めてセルフプロデュース。 ( 現在は世界を視野に入れた音楽の追求に励んでいます ! )

公式サイト

タローバンド

お酒と音楽をこよなく愛するメンバーが梅ヶ丘のバーfuzzで意気投合し結成。これまで複数の障害者施設でのイベント参加、バーでの演奏やサポートなど、和気藹々とした活動をしています。お子さんからオヤジ世代まで幅広い年齢層のオーディエンスと一緒に歌って踊って楽しめるステージを展開しています！
